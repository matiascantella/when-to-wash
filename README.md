# When to Wash

Electricity timing. Made simple.

## Setup

```bash
cd /Users/nadieestaaca/Desktop/when-to-wash
npm install
npm run dev
```

## MVP Features

✅ Brutalist design (white/gray background, minimal colors)
✅ Current status display (Cheap/Normal/Expensive)
✅ Best time today with start time and duration
✅ Timeline visualization of hourly prices
✅ Average price indicator
